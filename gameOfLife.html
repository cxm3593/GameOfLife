Conway's Game of Life

The Web Game Sample

This is a simple web based game of life written in Javascript. It starts with random living cells on a grid. The grid size is 400 by 400. And the random generation only starts on the central 200 by 200 square.

Some Pattern lives longer and some lives shorter. Since the patterns are randomly generated, most of the time, it will not stay in motion for a long time. You may try to refresh this page to get the grid updated.

If you are interested how it is implemented, navigate to the Implementation page for more information.
